# VAIXS - Sitio Web Corporativo

Bienvenido al repositorio de **VAIXS**, para una empresa de desarrollo de software. El sitio combina una interfaz atractiva con animaciones sutiles para mostrar servicios, productos y la propuesta de valor de la compañía.

Características
- Hero dinámico: Texto principal con efecto de slides (3 variantes) que destacan las soluciones digitales.
- Animación de lenguajes: Fondo con nombres de tecnologías (JavaScript, React, Python, etc.) que flotan horizontalmente, creado con canvas.
- Efecto máquina de escribir: Frase "Tu competencia ya se digitaliza. ¿Y tú?" con tipografía monoespaciada.
- Diseño responsive: Adaptado a móviles, tablets y desktop.
- Navegación suave: Scroll suave a secciones (Servicios, Productos, Contacto).
- Botón de WhatsApp flotante: Con chat de bienvenida.

Tecnologías
- React 18
- Tailwind CSS
- Canvas API para animaciones
- Hooks (useState, useEffect)
